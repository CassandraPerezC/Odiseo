OBJETIVOS




OBJETIVO GENERAL



Desarrollar un sistema informático para la administración y control empresarial IOFractal y las distintas empresas que se
ubican en el país, que permita hacer la contabilidad, recursos materiales, recursos financieros el área de recursos humanos,
así como un sistema de alertas que recuerde tareas y de un seguimiento de las mismas en forma rápida y correcta.

OBJETIVOS ESPECÍFICOS


1. Realizar una investigación preliminar para el desarrollo del sistema informático.
2. Realizar un análisis de requerimientos para el diseño y construcción del sistema informático.
3. Diseñar un sistema informático para la administración y control empresarial de IOFractal y las diferentes empresas mexicanas e internacionales.
4. Programar el sistema informático para la administración y control empresarial de IOFractal y las empresas que lo requieran.
5. Diseñar y ejecutar las pruebas pertinentes para comprobar el correcto funcionamiento del sistema para la administración y control empresarial de IOFractal y las empresas que lo requieran.
